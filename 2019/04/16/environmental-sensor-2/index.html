нҷҳкІҪм„јм„ң кө¬м„ұн•ҙліҙкё°(2)

мӨҖл№„л¬ј

  • лқјмҰҲлІ лҰ¬нҢҢмқҙ
  • PMS5003 (лЁјм§Җм„јм„ң)
  • DHT11 (мҳЁмҠөлҸ„м„јм„ң)
  • лқјмҰҲлІ лҰ¬нҢҢмқҙ 7мқём№ҳ н„°м№ҳмҠӨнҒ¬лҰ°
  • nodeMCU
  • 1.44 tft нҢЁл„җ

лӘ©м°Ё

  1. лқјмҰҲлІ лҰ¬нҢҢмқҙ SETTING
  2. SENSOR CONNECTION
  3. SENSOR VALUE к°Җм ёмҳӨлҠ” л°©лІ• л°Ҹ мҪ”л“ң
  4. SENSOR VALUEлҘј нҷ”л©ҙмңјлЎң лқ„мҡ°лҠ” л°©лІ•
  5. н”Ңлһ«нҸјмңјлЎң лҚ°мқҙн„° м „мҶЎл°©лІ•
  6. н”Ңлһ«нҸј лҚ°мқҙн„°лҘј мҠӨл§ҲнҠё лҜёлҹ¬м—җ лқ„мҡ°кё°

лқјмҰҲлІ лҰ¬нҢҢмқҙ + лҜём„ёлЁјм§Җ & мҳЁмҠөлҸ„м„јм„ң

лқјмҰҲлІ лҰ¬нҢҢмқҙ 3b мқҳ н•Җ л§өмқҖ м•„лһҳмҷҖ к°ҷмҠөлӢҲлӢӨ. лқјмҰҲлІ лҰ¬нҢҢмқҙ лӢӨлҘё лІ„м „мқ„ мӮ¬мҡ©н•ҳкІҢлҗңлӢӨл©ҙ м•„лһҳ н•Җл§өкіј л№„көҗн•ҳм—¬ к°ҷмқҖ Pin м—җлӢӨк°Җ м—°кІ° н•ҳмӢңл©ҙ лҗ©лӢҲлӢӨ.

м•„лһҳ мқҙлҜём§ҖлҠ” PM5003мқҳ PinMap мһ…лӢҲлӢӨ.
мӮ¬мҡ©н•ҳлҠ” н•ҖмқҖ 1,2,4,5 лІҲ н•Җмқ„ мӮ¬мҡ©н•  мҳҲм •мһ…лӢҲлӢӨ. лӮҳлЁём§Җ н•ҖмқҖ мӮ¬мҡ©н•ҳм§Җ м•Ҡкё° л•Ңл¬ём—җ м„ мқ„ м—°кІ° н•  кІҪмҡ° лӘЁл“  м„ мқ„ м—°кІ° н•ҳм§Җ л§җкі  4к°ңмқҳ м„ л§Ң м—°кІ°н•ҳм—¬ мӮ¬мҡ© н•  мҲҳ мһҲлҸ„лЎқ к°ңлҹүмқҙ н•„мҡ”н•©лӢҲлӢӨ.

DHT11 мқҖ л”°лЎң н•Җл§өмқ„ мһ‘м„ұн•ҳм§Җ м•ҠкІ мҠөлӢҲлӢӨ. м ңк°Җ к°Җм§Җкі  мһҲлҠ” DHT11мқҳ кІҪмҡ° м„јм„ңм—җ GND, VCC, DATA н•Җмқҙ м–ҙл””мқём§Җ н‘ңмӢңлҗҳм–ҙ мһҲкі , лҳҗн•ң DHT11 м„јм„ңмқҳ кІҪмҡ° л§Өмҡ° л§ҺмқҖ м ңмЎ°мӮ¬к°Җ мһҲкё° л•Ңл¬ём—җ мһҗмӢ мқҙ мӮ¬мҡ©н•ҳлҠ” м ңмЎ°мӮ¬мқҳ Pin Mapмқ„ м°ёкі  н•ҳмӢңкё° л°”лһҚлӢҲлӢӨ. лҢҖл¶Җл¶„ 3к°ңм—җм„ң 4к°ңмқҳ н•Җмқҙ мЎҙмһ¬н•ҳлҠ”лҚ° м ҖлҠ” мң„мқҳ 3к°ңмқҳ н•Җл§Ң мӮ¬мҡ©н•©лӢҲлӢӨ.

лқјмҰҲлІ лҰ¬ нҢҢмқҙмҷҖ м„јм„ң м—°кІ°

лқјмҰҲлІ лҰ¬ нҢҢмқҙмҷҖ м„јм„ң м—°кІ°мқҖ мң„мқҳ ліҙмқҙлҠ” н•Җм—җлӢӨк°Җ м—°кІ°н•ҙ мЈјмӢңл©ҙ лҗ©лӢҲлӢӨ.
PMS5003 м„јм„ңмқҳ 1,2 лІҲ PINмқҖ VCC, GNDмқҙкё° л•Ңл¬ём—җ лқјмҰҲлІ лҰ¬ нҢҢмқҙмқҳ VCC, GND мӣҗн•ҳмӢңлҠ” кіім—җ м—°кІ° н•ҙ мЈјмӢңл©ҙ лҗ©лӢҲлӢӨ.
DHT11 м„јм„ңмқҳ VCC, GND лҸ„ л§Ҳм°¬к°Җм§ҖлЎң мӣҗн•ҳлҠ” кіім—җ м—°кІ°н•ҙ мЈјл©ҙ лҗ©лӢҲлӢӨ.
лқјмҰҲлІ лҰ¬ нҢҢмқҙмқҳ VCC, GND м–ҙл””м—җ м—°кІ°н•ҙлҸ„ мғҒкҙҖ м—ҶмҠөлӢҲлӢӨ.

лқјмҰҲлІ лҰ¬ нҢҢмқҙ 7мқём№ҳ м—°кІ°н•ҳлҠ” л°©лІ•
Youtube л§ҒнҒ¬ мһ…лӢҲлӢӨ.

лқјмҰҲлІ лҰ¬ нҢҢмқҙ м „мӣҗ мҡ©лҹүмқҙ л¶ҖмЎұн•  мҲҳлҸ„ мһҲмңјлӢҲ м¶©л¶„н•ң м „л Ҙ мҡ©лҹүмқ„ к°Җм§„ м–ҙлҢ‘н„°лҘј мӮ¬мҡ©н•ҳмӢңкёё к¶ҢмһҘн•©лӢҲлӢӨ.

NodeMCU + Sensor

лҜёлӢҲ Arduino лқјкі  л¶ҲлҰ¬лҠ” NodeMCUмҷҖ Sensor м—°кІ° л°©лІ•мһ…лӢҲлӢӨ.
лқјмҰҲлІ лҰ¬нҢҢмқҙлЎң лӘЁл‘җ кө¬м„ұн•ҙлҸ„ лҗҳм§Җл§Ң м„јм„ңк°Җ лҠҳм–ҙлӮ  кІҪмҡ° Touch panelмқҙ к°ҖкІ©лҢҖк°Җ мғҒлӢ№н•ҳкё° л•Ңл¬ём—җ к°ҖкІ©м Ғмқё л©ҙмқҙ нҡЁмңЁм„ұмқҙ л–Ём–ҙм§‘лӢҲлӢӨ.
лҳҗн•ң, DashBoard лҘј л§Ңл“Ө мҳҲм •мқҙкё° л•Ңл¬ём—җ мӮ¬мӢӨмғҒ PanelмқҖ н•„мҡ”м—Ҷкі  м„јм„ңм—җм„ң н”Ңлһ«нҸјмңјлЎң м§Ғм ‘ лҚ°мқҙн„°лҘј мҸҙмЈјкё°л§Ң н•ҳл©ҙ лҗҳкё° л•Ңл¬ём—җ Deviceм—җм„ң Wifi к°Җ лҗҳлҠ” NodeMCUлҘј мӮ¬мҡ©н•ҳкІҢ лҗҳм—ҲмҠөлӢҲлӢӨ.

NodeMCUмқҳ н•Җл§өмқҖ мң„мҷҖ к°ҷмҠөлӢҲлӢӨ.

мң„мҷҖ к°ҷмқҙ OLED DHT DUST м—°кІ°н•ҙ мЈјмӢңл©ҙ лҗ©лӢҲлӢӨ.
Software мҪ”л“ңлҠ” лӢӨмқҢ мұ•н„°м—җм„ң лӢӨлЈЁлҸ„лЎқ н•ҳкІ мҠөлӢҲлӢӨ.

мқҙнӣ„ н•ҙм•јлҗ  мһ‘м—…

Sensor к°’мқ„ RaspberryPi лЎң к°Җм ёмҳӨлҠ” л¶Җл¶„кіј NodeMCUлЎң к°Җм ёмҳӨлҸ„лЎқ н•ҙм•јн•©лӢҲлӢӨ.
RaspberryPiлҠ” м§Ғм ‘м ҒмңјлЎң нҷ”л©ҙм—җм„ң м„јм„ң лҚ°мқҙн„°лҘј к°Җм ёмҳ¬ мҲҳ мһҲлҸ„лЎқ н•ҳкі , NodeMCUлҠ” н”Ңлһ«нҸјмңјлЎң м§Ғм ‘
м „мҶЎн•  мҲҳ мһҲлҸ„лЎқ н•©лӢҲлӢӨ.
